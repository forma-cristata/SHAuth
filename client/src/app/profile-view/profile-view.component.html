How to use this application

This tool helps you verify your GitHub commit IDs (SHAs) for your class assignments in a simple, straightforward way.
To begin using this tool, enter your GitHub username in the input field on the home page. Once logged in, you'll see a list of your classes to choose from. From there, choose the assignment you need to verify. Finally, enter your commit SHA in the verification field to check its validity.
After submitting your SHA, the system will analyze your commit and provide one of several possible responses. The form input box will glow green to indicate a valid commit SHA, or red if there's an issue. Additional details about your submission will appear below the input box.
To ensure a smooth verification process, always double-check that you're using your correct GitHub username (not your associated email). Make sure you've selected the appropriate class and assignment from the options provided. It's also crucial to verify that you've copied your SHA accurately to avoid any verification errors.
Once you have validated that your commit ID is valid, you may submit your project to Canvas.
